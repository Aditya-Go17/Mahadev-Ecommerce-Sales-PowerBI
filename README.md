# Mahadev Ecommerce Sales Dashboard

## Project Overview
This project provides a comprehensive analysis of e-commerce sales data using Power BI. [cite_start]It tracks key performance indicators (KPIs) like Total Amount, Quantity, AOV, and Profit.

## Dashboard Preview
![Dashboard Screenshot](dashboard_preview.jpg)

## Key Insights
* [cite_start]**Top Category:** Clothing dominates sales at 62.62% of total quantity.
* [cite_start]**Payment Preference:** Cash on Delivery (COD) is the most popular payment method (43.74%).
* [cite_start]**Regional Performance:** Maharashtra leads in sales amount among the tracked states.
* [cite_start]**Profitability:** Printers and Sarees are significant contributors to sub-category profit.

## Tools Used
* **Power BI:** Data Visualization & DAX
* **Excel/CSV:** Data Source
